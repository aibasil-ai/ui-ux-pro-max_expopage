# UI UX Pro Max 展示系統

這個專案會把 `ui-ux-pro-max` skill 內的風格、配色、字體、圖表與規則資料，整理成可瀏覽、可比較、可複製 Prompt 的 Astro 展示網站。

## 技術棧

- Astro
- Tailwind CSS
- React Islands
- Vitest
- Playwright

## 開發指令

```bash
npm install
npm run build:data
npm run dev
```

## 驗證指令

```bash
npm run test
npm run test:e2e
npm run build
```

## 資料來源

主要來源位於 `.codex/skills/ui-ux-pro-max/data/`。
